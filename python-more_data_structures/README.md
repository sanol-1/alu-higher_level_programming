# python-more_data_structures

This project covers more advanced usage of Python data structures: lists,
sets, and dictionaries, along with functional tools like `map`.

## Learning Objectives

- Working with lists, sets, and dictionaries
- Using `map` for transformations without loops
- Manipulating dictionary keys and values
- Solving a classic algorithmic problem (Roman numeral conversion)

## Files

| File | Description |
|------|-------------|
| `0-square_matrix_simple.py` | Squares all integers of a matrix |
| `1-search_replace.py` | Replaces all occurrences of an element in a list |
| `2-uniq_add.py` | Adds all unique integers in a list |
| `3-common_elements.py` | Returns common elements between two sets |
| `4-only_diff_elements.py` | Returns elements present in only one set |
| `5-number_keys.py` | Returns the number of keys in a dictionary |
| `6-print_sorted_dictionary.py` | Prints a dictionary by ordered keys |
| `7-update_dictionary.py` | Replaces or adds key/value in a dictionary |
| `8-simple_delete.py` | Deletes a key in a dictionary |
| `9-multiply_by_2.py` | Multiplies all dictionary values by 2 |
| `10-best_score.py` | Returns the key with the biggest value |
| `11-multiply_list_map.py` | Multiplies list values using map |
| `12-roman_to_int.py` | Converts a Roman numeral to an integer |
