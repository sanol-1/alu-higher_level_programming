# python-data_structures

This project covers Python's basic data structures: lists, tuples, and
matrices (lists of lists). It focuses on manipulating these structures
in a "C-like" and safe manner, without relying on exceptions or
built-in shortcuts where restricted.

## Learning Objectives

- What are lists and how to use them
- What are the differences and similarities between strings and lists
- What is a tuple
- Why lists and tuples are important
- What is a matrix
- How to slice a list or a tuple
- How to access an element of a list, negatively or positively
- How to iterate on lists
- How to use lists methods (append, insert, etc.)
- How to iterate into a list of tuples

## Files

| File | Description |
|------|-------------|
| `0-print_list_integer.py` | Prints all integers of a list |
| `1-element_at.py` | Securely retrieves an element from a list by index |
| `2-replace_in_list.py` | Replaces an element of a list at a specific position |
| `3-print_reversed_list_integer.py` | Prints all integers of a list in reverse order |
| `4-new_in_list.py` | Replaces an element in a copy of a list, without modifying the original |
| `5-no_c.py` | Removes all `c` and `C` characters from a string |
| `6-print_matrix_integer.py` | Prints a matrix of integers |
| `7-add_tuple.py` | Adds two tuples element-wise |
| `8-multiple_returns.py` | Returns the length and first character of a string |
| `9-max_integer.py` | Finds the biggest integer of a list |
| `10-divisible_by_2.py` | Finds all multiples of 2 in a list |
| `11-delete_at.py` | Deletes the item at a specific position in a list |
| `12-switch.py` | Switches the values of two variables |
