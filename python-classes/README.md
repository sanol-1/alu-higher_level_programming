# python-classes

This project covers the fundamentals of Object-Oriented Programming in
Python: classes, private attributes, properties (getters/setters), and
instance methods — built incrementally through a `Square` class.

## Learning Objectives

- How to define a class in Python
- Private instance attributes and name mangling
- Type and value validation with `TypeError` and `ValueError`
- Properties: getters and setters
- Writing instance methods (`area`, `my_print`)

## Files

| File | Description |
|------|-------------|
| `0-square.py` | Empty class `Square` |
| `1-square.py` | `Square` with a private `size` attribute (no validation) |
| `2-square.py` | `Square` with `size` type/value validation |
| `3-square.py` | `Square` with an `area()` method |
| `4-square.py` | `Square` with `size` getter/setter properties |
| `5-square.py` | `Square` with `my_print()` |
| `6-square.py` | `Square` with `size` and `position` properties |
