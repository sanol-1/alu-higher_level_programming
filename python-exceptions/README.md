# python-exceptions

This project covers exception handling in Python: `try`/`except`/`finally`,
raising custom exceptions, and safely handling type and range errors.

## Learning Objectives

- Why exceptions are important
- How to correctly handle an exception
- What is the purpose of catching exceptions
- How to raise a builtin exception
- When do we need to implement a clean-up action after an exception

## Files

| File | Description |
|------|-------------|
| `0-safe_print_list.py` | Prints x elements of a list |
| `1-safe_print_integer.py` | Safely prints a value as an integer |
| `2-safe_print_list_integers.py` | Prints and counts integers in a list |
| `3-safe_print_division.py` | Divides two integers with debug output |
| `4-list_division.py` | Divides two lists element by element |
| `5-raise_exception.py` | Raises a TypeError exception |
| `6-raise_exception_msg.py` | Raises a NameError exception with a message |
