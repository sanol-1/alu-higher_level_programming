# python-if_else_loops_functions

This project covers conditional statements, loops, and functions in
Python: writing readable control flow, using `for` loops with
`range()`, string formatting with `.format()`, and defining reusable
functions.

## Learning Objectives

- Why indentation is so important in Python
- How to use if, if...else statements
- How to use comparison operators
- How to use `for` and `while` loops
- How to use `range()`
- How to write functions
- Why you should not import any module you don't need
- What does "you don't need to understand" mean when the exercise
  provides code you should not touch

## Files

| File | Description |
|------|-------------|
| `0-positive_or_negative.py` | Prints whether a random number is positive, negative, or zero |
| `1-last_digit.py` | Prints the last digit of a random number and classifies it |
| `2-print_alphabet.py` | Prints the ASCII lowercase alphabet |
| `3-print_alphabt.py` | Prints the ASCII lowercase alphabet, excluding q and e |
| `4-print_hexa.py` | Prints numbers from 0 to 98 in decimal and hexadecimal |
| `5-print_comb2.py` | Prints numbers from 00 to 99 |
| `6-print_comb3.py` | Prints all unique combinations of two different digits |
| `7-islower.py` | Checks whether a character is lowercase |
| `8-uppercase.py` | Prints a string in uppercase |
| `9-print_last_digit.py` | Prints and returns the last digit of a number |
| `10-add.py` | Adds two integers |
| `11-pow.py` | Computes a to the power of b |
| `12-fizzbuzz.py` | Prints the FizzBuzz sequence from 1 to 100 |
